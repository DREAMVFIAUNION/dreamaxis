# DreamAxis Code of Conduct

We want DreamAxis to be a practical, respectful, and welcoming open-source project.

## Our expectations

Please:

- be respectful and constructive
- assume good intent before escalating
- focus feedback on the work, not the person
- welcome contributors with different backgrounds and experience levels
- share context when disagreeing
- help keep discussions useful for future readers

## Unacceptable behavior

This project does not allow:

- harassment or intimidation
- hateful or discriminatory language
- doxxing or sharing private information without permission
- sexualized language or unwanted sexual attention
- personal attacks, insults, or bad-faith pile-ons
- spam, trolling, or deliberately disruptive behavior

## Maintainer responsibilities

Project maintainers may:

- moderate discussions, issues, and pull requests
- request changes in tone or behavior
- remove abusive or off-topic content
- temporarily or permanently restrict participation when needed

## Scope

This applies to project spaces including:

- GitHub issues
- pull requests
- discussions
- documentation discussions
- any community spaces explicitly used for DreamAxis collaboration

## Reporting

If you experience or witness harmful behavior:

1. avoid escalating in public if that would make the situation worse
2. contact the maintainers through the private security/support path described in [SUPPORT.md](./SUPPORT.md) or [SECURITY.md](./SECURITY.md)
3. include links, screenshots, or other relevant context if possible

Reports will be reviewed in good faith and handled as discreetly as practical.

## Enforcement

Maintainers may take any action they believe is appropriate to protect the project and its contributors, including warning, content removal, temporary suspension, or permanent ban from project spaces.

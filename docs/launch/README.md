# DreamAxis Launch Pack

This folder contains the English-first public launch copy for DreamAxis.

## Included assets

- `release-announcement.md` - GitHub release/discussion announcement copy
- `x-short-post.md` - single short X/Twitter launch post
- `x-thread.md` - longer launch thread
- `community-launch-post.md` - longer post for forums, Discord, Reddit, or Hacker News-style sharing
- `intro-blurb.md` - short reusable project blurb for profiles, directories, and forms
- `seeded-issues.md` - first public issue seeds aligned with `v0.1.x polish`

## Positioning rules

All launch copy should keep the same framing:

- DreamAxis is a **local-first agent execution platform**
- the default path is **no-signup**
- users keep **their own keys and data**
- the product already supports **CLI Runtime + Browser Runtime**
- the system includes **skill packs + knowledge packs**
- the current release is a foundation, not a finished multi-agent operating system

# DreamAxis short intro blurbs

## Recommended

DreamAxis is a local-first, open-source agent execution platform for self-hosted AI workflows. It combines OpenAI-compatible provider connections, CLI + Browser runtimes, skill packs, and knowledge packs into a reusable execution layer that goes beyond chat alone.
